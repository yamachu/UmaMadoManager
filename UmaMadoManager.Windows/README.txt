UmaMadoManager

## 使い方

1. UmaMadoManager.Windows.exeを右クリックし、「管理者として実行」をクリック
2. 環境によっては「WindowsによってPCが保護されました」と表示されます。このアプリケーションが信用できるのであれば、そのモーダル内の詳細情報をクリックし、実行をクリック。
3. Iconがタスクトレイに表示されるので、右クリックし設定を行う。ゲーム画面が横長の時に伸びてほしい場合はHorizontalをマウスホバーし、その下のFullを選択。縦長の時は同様にVerticalの箇所を選択してください。
4. DMM版 ウマ娘プリティーダービー を起動

起動する度に「管理者として実行」の処理を行うのが煩わしい場合は、アプリケーションへのショートカットを作成し、管理者として実行を行うように変更すると便利です。

## アンインストール方法

ディレクトリに展開するだけで使えるアプリケーションのためアンインストールは解凍したディレクトリの削除だけで完了します。
設定ファイルは
C:\Users\%username%\AppData\Local\UmaMadoManager.Windows
に存在するディレクトリごと削除することで完全な削除が完了します。
